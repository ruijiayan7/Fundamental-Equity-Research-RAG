# 0-前置条件

安装wsl，参考：[https://learn.microsoft.com/zh-cn/windows/wsl/install](https://learn.microsoft.com/zh-cn/windows/wsl/install)



安装docker，参考：[https://learn.microsoft.com/zh-cn/windows/wsl/tutorials/wsl-containers](https://learn.microsoft.com/zh-cn/windows/wsl/tutorials/wsl-containers)



安装nltk数据包，参考：[https://blog.csdn.net/qq_43140627/article/details/103895811](https://blog.csdn.net/qq_43140627/article/details/103895811)

阿里云大模型调用，参考：[https://bailian.console.aliyun.com/?tab=model#/model-market]https://bailian.console.aliyun.com/?tab=model#/model-market



准备镜像，国内镜像源：[https://github.com/dongyubin/DockerHub](https://github.com/dongyubin/DockerHub)

启动wsl，下载镜像

镜像源使用方式：原镜像名称之前加上镜像源地址

docker pull postgres:15-alpine

docker pull docker.1ms.run/postgres:15-alpine

镜像重命名，与docker-compose文件中的镜像保持一致即可
docker tag docker.1ms.run/postgres:15-alpine postgres:15-alpine




# 后端启动

cd backend

## 步骤1：需下载nltk包，修改docker-compose.yml文件中nltk的挂载路径

## 步骤2：修改.env文件中 DASHSCOPE_API_KEY

## 启动后端服务

docker compose up -d --build

查看后端日志：docker logs -f swxy_api


\# 前端启动

cd frontend

npm install

npm run dev
